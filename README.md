# ALPR

The following ALPR system goes through 4 basic steps. Encompassing the entirety of any automatic number plate recognition.

1) Photo Acquisition
2) Localization
3) Segmentation
4) Recognition

# Packages & Libraries
    opencv
    argparse
    numpy
    pickle
    imutils
    skimage
    sklearn
    glob
